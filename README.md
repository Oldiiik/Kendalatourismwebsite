  # Kendala Tourism Website

  KenDala is a next-generation digital travel guide designed for international visitors who want to explore Kazakhstan beyond standard tourist routes. The project is built around a simple idea: to make traveling    across the country clear, accessible, and culturally meaningful. Instead of forcing travelers to search for scattered information across different sources, KenDala brings everything together in one place — from   destinations and ready-made travel routes to practical advice and cultural context that truly matters for someone visiting Kazakhstan for the first time.

  The platform focuses not only on major cities but also on regional diversity. Kazakhstan is vast and varied, and KenDala helps users discover that contrast: mountains and canyons, steppes and lakes, historical    landmarks and modern urban spaces. Travelers can explore locations through an interactive map, find both well-known attractions and lesser-known spots, plan trips based on their interests and time, and get        clear guidance on transportation, logistics, and everyday travel details.

  A key part of the project is cultural understanding. KenDala includes essential information about traditions, national cuisine, local etiquette, and social norms, helping visitors avoid common misunderstandings   and engage with the culture in a more respectful and informed way. It also provides practical insights such as how to move between cities, useful phrases, and what to expect when traveling in different            regions of the country.

  The mission of KenDala is to make Kazakhstan more visible and approachable for the global travel community, presenting it as a rich and diverse destination while supporting regional tourism. It is not just a      collection of places, but an attempt to create a complete travel experience where technology helps people better understand the country, its landscapes, and its culture.

  The project is currently evolving, with new locations, routes, and content being added over time, aiming to become a reliable starting point for anyone planning a journey across Kazakhstan.

  The original project is available at https://kendala.figma.site/

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
